## 开发建议

1. **从简单到复杂推进：** 先完成 ★~★★ 任务（联系我们、公告、留言板、文档），积累开发节奏
2. **证书识别**可先接入百度 OCR（免费额度充足），快速验证流程，后续切换大模型视觉 API
3. **项目管理智能体**建议分三阶段：
   - 第一阶段：数据模型 + 项目/任务 CRUD + 看板视图
   - 第二阶段：周报系统 + 定时提醒
   - 第三阶段：AI 需求解析 + 周报分析
4. **CLI 接口**建议在项目管理 API 稳定后再开发，避免接口频繁变更
5. AI 功能建议统一封装为 `ai_service.py`，便于切换不同大模型提供商
